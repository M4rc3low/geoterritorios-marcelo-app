# GeoTerritórios no portfólio

## Resumo curto

GeoTerritórios é uma aplicação web para organizar territórios, prédios, apartamentos, regiões e indicadores de cobertura, com foco em produtividade e controle visual.

## Problema identificado

O controle territorial pode ficar espalhado em mapas, planilhas, anotações e conferências manuais. Isso dificulta saber o que já foi feito, o que ainda falta e quais áreas precisam de prioridade.

## Solução proposta

Criar uma base digital organizada para acompanhar territórios, visualizar progresso, controlar pendências e preparar futuras integrações com mapas e dados geográficos.

## Diferenciais do projeto

- Projeto baseado em necessidade real
- Potencial de uso com mapas e geodados
- Organização de dados territoriais
- Visão operacional e prática
- Base para dashboard de cobertura

## Tecnologias demonstradas

- React
- Vite
- Tailwind CSS
- React Leaflet
- Recharts
- Componentização
- Visualização de dados
